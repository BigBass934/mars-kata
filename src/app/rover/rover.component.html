INFORMATION

This is an implementation of the Mars Rover Kata, with a UI implementation using Angular, Typescript, & OOPs

====================================

NOTES:

- Realistic "planet-wrapping": The rover will move "around" the spherical planet

- Obstacle Detection: The rover will stop & give an alert when attempting to move onto a coordinate with an obstacle.

*For ease of use, the "opposite side" of the planet is clear of obstacles

INSTRUCTIONS

Input commands to move the rover, using either the buttons or by entering a string of them manually into the input field below:

"f": Move Forward,

"b": Move Backward,

"l" Rotate Left,

"r": Rotate Right

"Reset Rover" resets the rover position, but not the simulation. "Reset Simulation" resets the rover and creates a new planet
